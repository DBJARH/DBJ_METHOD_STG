<details markdown="1">
<summary><b>GH WWW Publishing Terminology</b></summary>
<p>

- What is `github.io`?
  - `github.io` is the domain for **GitHub Pages**, a service that automatically turns content stored in a GitHub repository into a live website. It acts as the "host" that tells the internet to look inside the `ironcodelabs-com` account to find the files for the **EA** and **EA_STG** projects.
- What is `https:`?
  - The `https:` at the start of a web address means the connection is private and protected by **encryption**. This keeps information safe from eavesdropping.
- Staging vs. Production
  - **Staging** is a "rough draft" version of the website where new content is reviewed before going live. **Production** is the final, live version ready for the public.

</p>
</details>
